ScalaExtractorsLib
==================

A library of Scala Extractors for common scenarios.

To run the tests issue:
	lib/test

To run the samples issue:
	samples/run

Extractors supported (or planned)

net/
	IP4		Internet Address v4
	IP6		Internet Address v6 (wip)
	Domain
	Email

io/
	File
	WinFile

biz/
	IBAN
	
biz/it/
	CodiceFiscale

crypto/
	SHA1
	MD5
